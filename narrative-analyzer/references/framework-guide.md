# 舆情叙事框架分析参考手册

本文档为 narrative-analyzer Skill 提供深度参考。当 LLM 在框架识别中遇到模糊情况时，可读取本文件获取更精确的判断指导。

---

## 一、五种叙事框架深度定义

### 1. 冲突框架 (Conflict Frame)

**学术定义**：强调不同个体、群体或机构之间的冲突、分歧和对立，将事件呈现为两方或多方的博弈。

**识别关键词**：炮轰、对峙、争论、反击、互怼、撕裂、对抗、矛盾、交锋、分歧、角力、博弈、口水战、针锋相对

**典型场景**：某企业与监管部门就罚款决定公开争执——报道强调双方立场对立，使用"回击""不服"等措辞。

**区分要点**：如果报道虽然涉及两方但重点是描述经济损失而非对抗过程，应归为经济后果框架而非冲突框架。

### 2. 人情味框架 (Human Interest Frame)

**学术定义**：通过个人故事、情感化叙事或人性化视角呈现事件，激发受众情感共鸣。

**识别关键词**：感人、心碎、泪目、令人揪心、暖心、他/她的故事、亲历者、普通人、家属、幸存者、背后的故事、人间冷暖

**典型场景**：自然灾害报道中聚焦一位母亲寻找孩子的经历，而非灾害整体损失统计。

**区分要点**：如果报道用个人故事来指出制度缺陷（"她的遭遇暴露了XX漏洞"），主框架是责任归因，人情味为辅框架。

### 3. 责任归因框架 (Attribution of Responsibility Frame)

**学术定义**：将事件的原因或解决方案归因于特定主体（政府、企业、个人等），引导受众判断"谁该负责"。

**识别关键词**：谁之过、责任、监管缺失、失职、问责、追责、推诿、甩锅、负责人、根源在于、管理漏洞、制度缺陷

**典型场景**：食品安全事件中，报道直指"监管部门长期不作为是根本原因"。

**区分要点**：如果报道只是客观描述"政府回应了此事"但未暗示政府应负责，不算责任归因框架，可能是中立陈述。

### 4. 经济后果框架 (Economic Consequence Frame)

**学术定义**：以经济、财务或物质利益的视角呈现事件，强调事件的经济影响和后果。

**识别关键词**：损失、亿元、股价、暴跌、市值蒸发、经济影响、成本、赔偿、行业震荡、GDP、就业、供应链、涨价、降薪

**典型场景**：某科技公司数据泄露事件，报道聚焦于"公司市值一夜蒸发200亿"而非数据泄露的技术细节。

**区分要点**：如果报道提到经济数字但核心是追问"谁该赔偿"，主框架应归为责任归因，经济后果为辅。

### 5. 道德框架 (Morality Frame)

**学术定义**：从道德、伦理或价值观的角度评判事件，诉诸社会公序良俗和基本道德标准。

**识别关键词**：良心、底线、道德、伦理、价值观、公序良俗、天理难容、丧尽天良、是非、正义、善恶、人性、廉耻

**典型场景**：商家售卖假冒伪劣商品，评论聚焦"商业道德沦丧""做人的底线在哪里"。

**区分要点**：道德框架与责任归因框架容易混淆。关键区别：道德框架重"应不应该"（价值判断），责任归因框架重"谁导致了这件事"（因果分析）。

## 二、归因主体分类说明

| 归因主体 | 判断标准 | 易误判情况 |
|---------|---------|-----------|
| 政府 | 报道指向政策制定者、行政管理部门、官员 | "政策出了问题"指向制度而非政府本身时，归"制度" |
| 企业 | 报道指向具体公司、管理层、企业行为 | "行业普遍如此"指向行业惯例时，归"制度" |
| 个人 | 报道指向具体当事人的个人行为或决策 | "他被迫这样做"暗示制度因素时，考虑双归因 |
| 制度 | 报道指向规则、法律、行业惯例、体制机制 | 注意与"政府"区分：制度是规则本身，政府是执行者 |
| 技术 | 报道指向技术缺陷、算法问题、系统故障 | "算法歧视"：强调算法缺陷→技术；强调企业明知故犯→企业 |
| 社会 | 报道指向社会风气、公众行为、文化因素 | "内卷"可归社会（风气）或制度（结构性问题），看报道侧重 |
| 自然 | 报道指向自然灾害、不可抗力 | 灾后追责转向人为因素时，归因应转向政府/企业/制度 |
| 不明确 | 报道未明确指向任何责任主体 | 纯事实报道、客观描述多属此类 |

## 三、框架冲突度评估指南

| 冲突度 | 判断标准 | 量化参考 |
|-------|---------|---------|
| **低** | 不同来源框架类型基本一致，归因方向相同 | >70% 来源使用相同主框架 |
| **中** | 框架类型有分歧但归因方向大致一致 | 40%-70% 来源使用相同主框架 |
| **高** | 框架类型和归因方向都存在明显分歧 | <40% 来源使用相同主框架，或归因主体出现对立 |

**典型高冲突场景**：官方媒体用"责任归因→个人"框架，社交媒体用"责任归因→企业/制度"框架，同时民间还有"道德框架"的声音。

## 四、舆情阶段判断参考

| 阶段 | 核心特征 | 辅助判断指标 |
|------|---------|------------|
| **萌芽期** | 少量报道、低热度、单一来源 | 仅 1-2 个来源提及，热度值低，无评论讨论 |
| **升温期** | 多来源开始报道、热度上升 | 3+ 来源报道，热度显著增长，开始出现评论和讨论 |
| **爆发期** | 全平台覆盖、热搜上榜、强情感 | 百度/微博均上热搜，情感强度平均≥3.5，大量讨论 |
| **衰减期** | 热度下降、新信息减少 | 热度值较峰值下降 50%+，话题被新热点替代 |

**注意**：由于我们是单次采集快照（非持续监控），阶段判断基于当前数据特征推断，而非历史趋势对比。在只有单次数据的情况下：
- 高热度 + 多来源 + 强情感 → 倾向判断为"爆发期"
- 中等热度 + 多来源 → 倾向判断为"升温期"
- 低热度 + 少来源 → 倾向判断为"萌芽期"
