# 舆情叙事框架分析参考手册

## 第一章：五种叙事框架详解

### 1. 冲突框架（Conflict Frame）

学术定义：强调群体、机构或个人之间的矛盾、争议、博弈和公开对抗，是 Semetko 与 Valkenburg 框架分类中的典型冲突呈现方式。

识别关键词：炮轰、回怼、争议、对峙、反击、质疑、冲突、矛盾、对抗、互撕、激辩、交锋、针锋相对、拉锯、博弈。

典型举例：
- 标题：“某部门怒斥企业造假，企业强硬回应”
- 说明：报道重点是双方公开对抗和舆论张力，而不是谁最终负责。

易混淆区分：
- 与责任归因框架的区别：冲突框架突出“谁和谁吵起来了”，不一定明确责任判定。
- 与道德框架的区别：若文章主要谴责失德失范而非双方对抗，优先判为道德框架。

### 2. 人情味框架（Human Interest Frame）

学术定义：通过个体经历、情绪故事和具象人物命运，引导受众从共情视角理解公共事件。

识别关键词：令人心碎、感动、他的故事、温暖、泪目、平凡英雄、暖心、催泪、背后的故事、一个人的坚守、生命的重量、真实经历、逆行者、母亲、孩子。

典型举例：
- 标题：“外卖小哥风雪中骑行 30 公里只为准时送达”
- 说明：报道中心是人物处境和情感唤起，而不是平台规则本身。

易混淆区分：
- 若人物故事只是引出“监管为何缺失”“制度为何失灵”，则主框架可能转向责任归因。
- 若报道更强调双方对立和口水战，不要误判为人情味框架。

### 3. 责任归因框架（Attribution of Responsibility Frame）

学术定义：将事件结果、风险或伤害明确指向某个责任主体，核心问题是“谁该负责”。

识别关键词：谁之过、监管缺失、失职、问责、追责、责任、玩忽职守、管理漏洞、负主要责任、难辞其咎、不作为、违规、渎职、甩锅、追问。

典型举例：
- 标题：“多部门推诿，受害者维权三年无果”
- 说明：报道主轴是责任链条和问责对象，而非事件过程本身。

易混淆区分：
- 与冲突框架的区别：只要文本明确追问责任归属，优先视为责任归因框架。
- 与经济后果框架的区别：即便提到损失，若重心仍是“谁造成的”，仍属于责任归因。

### 4. 经济后果框架（Economic Consequence Frame）

学术定义：聚焦事件对个人、企业、行业或宏观经济带来的成本、收益和市场影响。

识别关键词：损失、股价、市值蒸发、成本、市场震荡、行业冲击、GDP、就业、消费、供应链、估值、经济影响、亏损、订单、投资。

典型举例：
- 标题：“某企业暴雷，供应链上下游数百亿资产面临风险”
- 说明：核心不是情绪或责任，而是经济波及范围和财务后果。

易混淆区分：
- 仅出现数字不等于经济后果框架，必须把经济影响作为报道主中心。
- 若文章主要在指责失责方，哪怕带有损失描述，也优先判责任归因框架。

### 5. 道德框架（Morality Frame）

学术定义：从伦理、价值观、公序良俗和社会善恶标准出发，对事件进行规范性评判。

识别关键词：底线、良心、价值观、伦理、道义、天理难容、人性、公德、无良、有悖常理、丧尽天良、道德沦丧、良知、羞耻、失范。

典型举例：
- 标题：“竟然向灾区卖高价水，良心何在？”
- 说明：报道重心是道德谴责和价值判断，而非单纯责任认定。

易混淆区分：
- 若文本核心句式是“某某应负责”，优先判责任归因框架。
- 若文本核心是“这种行为突破底线、不可接受”，优先判道德框架。

## 第二章：归因主体分类标准

| 归因主体 | 定义 | 示例 |
|---|---|---|
| 政府 | 政策制定者、执法机关、监管部门、公共管理机构 | “卫生部门监管不力” |
| 企业 | 涉事公司、品牌、平台、商业组织 | “某企业偷工减料” |
| 个人 | 明确指向具体自然人或个体行为 | “司机疲劳驾驶” |
| 制度 | 法规设计、治理机制、流程缺陷、结构性漏洞 | “现行法律存在漏洞” |
| 技术 | 算法、系统故障、模型偏差、平台技术设计 | “算法推荐导致信息茧房” |
| 社会 | 行业潜规则、群体心态、公共风气、集体习惯 | “行业潜规则盛行” |
| 自然 | 天灾、气候、地质、不可抗自然因素 | “百年一遇暴雨” |
| 不明确 | 报道未形成清晰责任指向，仅描述事实状态 | “多方仍在调查中” |

特殊情况说明：
- “算法歧视”：如果强调系统机制缺陷，归为技术；如果强调平台明知故犯，归为企业。
- “政策执行不到位”：如果批评政策设计本身，归为制度；如果指向执行部门失职，归为政府。
- “网友集体围攻”：如果强调网暴氛围和群体行为，归为社会；若聚焦具体带头人，归为个人。

## 第三章：框架冲突度评估

- 低：最高框架占比大于等于 70%，叙事口径比较一致。
- 中：最高框架占比在 40% 到 69% 之间，存在次级竞争框架。
- 高：最高框架占比低于 40%，不同来源叙事分歧明显。

## 第四章：快速判断建议

- 先看报道最想让读者关注什么，再选主框架，不要被个别词语误导。
- 如果一条报道同时包含多个特征，优先记录“主框架 + 次级信息”，不要强行多选。
- 归因主体优先选择最直接的责任落点；无法确定时使用“不明确”，不要臆造责任方。
- `core_argument` 尽量用短句概括，不超过 30 字，聚焦报道的中心主张。
